# MERN E-Commerce Store 👇

MERN E-Commerce Store is a full-stack web application built using MongoDB, Express.js, React.js, and Node.js.
It provides features such as product listing, category management, and dynamic product rendering from the database.
The application uses RESTful APIs with secure backend architecture and scalable database design.
Frontend is developed using React with modern UI components and API integration for real-time data display.
This project demonstrates full-stack development skills, database handling, and deployment-ready architecture.
